# JSONL 檢視與編輯工具 (SillyTavern 特化版)

## 專案概述
本工具為**純前端、單一 HTML 檔案**的輕量化對話紀錄檢視與編輯器。主要用於載入、美化、修改並匯出 SillyTavern（酒館）格式的 `.jsonl` 聊天紀錄檔案。工具內置雙重 ID 追蹤、分頁動態載入、正則美化包管理器、雲端 Gist 同步等進階後處理功能，並針對手機端進行了像素級的響應式設計（RWD）最佳化。

---

## 核心功能與規格

### 1. 資料處理與渲染
* **雙軌 ID 追蹤**：
  * **酒館 ID (`_local_id`)**：對話切分後的相對樓層 ID，由 `0` 開始重新計算。
  * **總計 ID (`_original_id`)**：對話切分前保留的絕對物理樓層順序，不受拆檔影響。
* **動態增量載入 (Infinite Scroll)**：利用 `IntersectionObserver` 進行底端滾動偵測，每次僅增量渲染 30 筆對話 (`BATCH_SIZE = 30`)，確保長篇對話在手機與電腦端皆能流暢執行。
* **內建編輯器**：支援將單一對話區塊切換為文字編輯區 (高度 400px)。儲存時同步更新記憶體主陣列及 `swipes`（多選分支）結構，維持資料結構完整性。

### 2. 文本轉譯與 Markdown 特化
* **精準換行還原**：透過 `marked.use({ breaks: true })` 將單次換行忠實轉譯為 `<br>`，完美對齊酒館原生介面及真實聊天軟體的對話排版體驗。
* **思考鏈 (Thought Chain) 抽離**：自動識別 `<thinking>` 標籤或 `reasoning` 欄位，將思考鏈抽離至獨立的次級字級摺疊區塊，並支援全域一鍵展開/收合。
* **心聲變色與進階語法**：
  * Markdown 單星號斜體 (`em`) 定義為角色心聲，並套用專屬的高識別度顏色 (`--thought-color`)。
  * 支援標準 Markdown 表格與 Mermaid 流程圖語法。

### 3. 響應式介面 (RWD) 與 UX 最佳化
* **智慧頂端控制列**：
  * 以自訂的「檔名標籤按鈕」取代原生醜陋的上傳組件。
  * **電腦版**：完整顯示長檔名，允許自動折行。
  * **手機版**：檔名寬度鎖死 `45vw` 並自動截斷 (`text-overflow: ellipsis`)。右側常駐「⬆ / ⬇」跳轉快捷鍵與「☰」面板開關。
* **樓層標頭 Meta 區塊**：
  * 強制統一實體高度 (38px)，消除按鈕與標籤高低差。
  * **手機版自適應雙排重構**：首排左側顯示雙色 ID 車票，右側顯示無文字的純圖示操作按鈕 (編輯、儲存、取消)；次排則獨立 100% 寬度顯示時間戳記與角色名稱。
* **懸浮側邊面板**：收納所有進階設定 (字級縮放、主題切換、跳轉、預設包與雲端同步等)，手機版預設隱藏不擋視線，點擊頂端選單後由右側平滑滑出。

### 4. 進階延伸擴充模組
* **視覺預設包管理器 (Regex Package Manager)**：
  * 允許自訂正則表達式 (Pattern) 與替換 HTML 範本 (Template)，將特定格式標籤 (如 `<profile>`) 轉化為摺疊方塊或自訂樣式。
  * 支援即時新增、刪除與 JSON 格式的完整匯入/匯出。
* **GitHub Gist 雲端儲存**：
  * 整合 GitHub API，透過輸入 Personal Access Token (PAT)，可一鍵於雲端建立私密 Gist。
  * 支援自訂正則預設包的雙向同步 (Upload 備份 / Fetch 下載)。
